# Frontend

Build with Angular. Doesn't look good on mobile (yet)

## Technologies used
- **Framework:** Angular v20
- **UI Library:** Bootstrap 5
- **CSS Preprocessor:** SCSS
- **Icons:** Font Awesome
- **Notifications**: Toastr

## Development
Prerequisites
- Node.js and npm
- Angular CLI

```
npm install
npm start
```

## Deployment
Deployed on [Cloudflare Pages](https://pages.cloudflare.com/)
